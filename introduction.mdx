## What is Ownstash Downloader?

Ownstash Downloader is a **desktop-first media downloader** built with Tauri (Rust backend + React frontend) that focuses on fast local downloads with built-in engine management for `yt-dlp` and `SpotDL`. Download media from 1000+ platforms including YouTube, Spotify, Instagram, Twitter, TikTok, and more—all stored securely on your local machine.

## Core Features

<AccordionGroup>
  <Accordion title="1000+ Platform Support via yt-dlp" icon="globe">
    Download from virtually any video or audio platform on the internet:
    
    - **Video Platforms**: YouTube, Vimeo, Dailymotion, Twitch, TikTok
    - **Social Media**: Twitter/X, Instagram, Facebook
    - **Audio Platforms**: SoundCloud, Mixcloud, Bandcamp
    - **File Hosting**: Google Drive, OneDrive, Dropbox, Mega, MediaFire
    - And 1000+ more platforms supported by yt-dlp

      The app automatically manages yt-dlp updates to ensure compatibility with the latest platform changes.
    </Note>

  <Accordion title="Spotify Downloads via SpotDL" icon="spotify">
    Download music from Spotify using SpotDL's intelligent matching system:
    
    - Downloads tracks, albums, playlists, and podcasts
    - Searches YouTube for the best audio match
    - Embeds metadata and album artwork automatically
    - Supports lyrics embedding
    - Handles large playlists with progress tracking
    
    <CodeGroup>
    ```typescript Example: Spotify Download Detection
    // From HomePage.tsx:214-237
    if (isSpotifyLink) {
        setIsLoading(true);
        setError(null);

        toast.info('🎵 Fetching Spotify info... This may take a moment as we search for the best audio match.', {
            duration: 5000,
        });

        try {
            const info = await api.getSpotifyInfo(targetUrl);
            await api.addSearch(targetUrl, info.title, info.thumbnail);
            setSpotifyMediaInfo(info);
            setShowSpotifyModal(true);
            toast.success(`Found: ${info.title}`);
        } catch (err) {
            const errorMsg = err instanceof Error ? err.message : 'Failed to fetch Spotify info';
            setError(errorMsg);
            toast.error(errorMsg);
        } finally {
            setIsLoading(false);
        }
    }
    ```
    </CodeGroup>
  </Accordion>

  <Accordion title="PIN-Protected Encrypted Vault" icon="shield-halved">
    Store sensitive downloads in an encrypted vault with military-grade security:
    
    - **AES-256-GCM encryption** for file protection
    - **Argon2 password hashing** for PIN security
    - Support for individual files and entire folders
    - Automatic compression for folder uploads
    - `.slasshy` encrypted file extension
    
    <CodeGroup>
    ```rust Vault Encryption Implementation
    // From vault.rs:1-27
    // Vault module for PIN-protected encrypted storage
    // Uses AES-256-GCM for file encryption and Argon2 for PIN hashing
    // Supports both single files and folder uploads (compressed to ZIP then encrypted)

    use aes_gcm::{
        aead::{Aead, KeyInit, OsRng},
        Aes256Gcm, Nonce,
    };
    use argon2::{
        password_hash::{PasswordHash, PasswordHasher, PasswordVerifier, SaltString},
        Argon2,
    };

    const VAULT_DIR_NAME: &str = "vault";
    const VAULT_CONFIG_FILE: &str = "vault_config.json";
    pub const ENCRYPTED_EXTENSION: &str = ".slasshy";
    const LEGACY_EXTENSION: &str = ".vault"; // For backward compatibility
    const NONCE_SIZE: usize = 12;
    const KEY_SIZE: usize = 32;
    ```
    </CodeGroup>
    
    <Warning>
      If you forget your vault PIN, there is no recovery method. Make sure to remember your PIN or store it securely.
    </Warning>
  </Accordion>

  <Accordion title="Download Queue & History" icon="list-check">
    Comprehensive download management with real-time progress tracking:
    
    - Real-time progress updates with speed and ETA
    - Download history with thumbnails and metadata
    - Cancel, retry, or delete downloads
    - Open download folder or play media in-app
    - Track downloads by platform and status
    
    <CodeGroup>
    ```typescript Real-time Progress Updates
    // From DownloadsPage.tsx:282-306
    // Set up yt-dlp progress listener
    api.onDownloadProgress((progress) => {
        setProgressMap(prev => {
            const newMap = new Map(prev);
            const previousProgress = prev.get(progress.id);
            const stabilized = stabilizeProgressEvent(progress, previousProgress);
            newMap.set(progress.id, stabilized);
            return newMap;
        });

        // Update download status in database if completed or failed
        if (progress.status === 'completed' || progress.status === 'failed' || progress.status === 'cancelled') {
            api.updateDownloadStatus(progress.id, progress.status);
            // Refresh downloads list
            setTimeout(loadDownloads, 500);
        }
    }).then(fn => {
        unlistenYtdlp = fn;
    }).catch(err => {
        console.error('[DownloadsPage] Failed to set up yt-dlp progress listener:', err);
    });
    ```
    </CodeGroup>
