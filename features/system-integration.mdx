## Overview

Ownstash integrates deeply with your operating system for a native desktop experience. Run in the background, get notified about downloads, and see progress directly in your taskbar.

## System Tray

Ownstash runs in the system tray for quick access and background operation.

### Tray Icon Setup

```rust
use tauri::tray::TrayIconBuilder;
use tauri::menu::{Menu, MenuItem};

// Create tray menu
let quit_i = MenuItem::with_id(
    app.handle(), 
    "quit", 
    "Quit", 
    true, 
    None::<&str>
).unwrap();

let show_i = MenuItem::with_id(
    app.handle(), 
    "show", 
    "Show", 
    true, 
    None::<&str>
).unwrap();

let hide_i = MenuItem::with_id(
    app.handle(), 
    "hide", 
    "Hide", 
    true, 
    None::<&str>
).unwrap();

let menu = Menu::with_items(
    app.handle(), 
    &[&show_i, &hide_i, &quit_i]
).unwrap();

let _tray = TrayIconBuilder::new()
    .menu(&menu)
    .icon(app.default_window_icon().unwrap().clone())
    .on_menu_event(move |app, event| {
        match event.id.as_ref() {
            "quit" => {
                app.exit(0);
            }
            "show" => {
                show_main_window(app);
            }
            "hide" => {
                close_main_window(app);
            }
            _ => {}
        }
    })
    .on_tray_icon_event(move |tray, event| {
        // Left-click tray icon to show/restore window
        if let TrayIconEvent::Click { 
            button: MouseButton::Left, .. 
        } = event {
            let app = tray.app_handle();
            show_main_window(&app);
        }
    })
    .build(app)?;
```

### Minimize to Tray

Keep Ownstash running in the background:

```rust
fn is_minimize_to_tray_enabled(app: &AppHandle) -> bool {
    let app_state = app.state::<AppState>();
    if let Ok(db) = app_state.db.lock() {
        return db
            .get_setting("minimize_to_tray")
            .unwrap_or(Some("false".to_string()))
            .unwrap_or("false".to_string())
            == "true";
    }
    false
}

// Handle window close event
.on_window_event(move |window, event| {
    if window.label() != MAIN_WINDOW_LABEL {
        return;
    }
    
    if let WindowEvent::CloseRequested { api, .. } = event {
        let should_minimize = is_minimize_to_tray_enabled(&window.app_handle());
        
        if should_minimize {
            // Hide window instead of closing
            api.prevent_close();
            let _ = window.hide();
        }
    }
});
```

### Window Management

```rust
const MAIN_WINDOW_LABEL: &str = "main";

fn ensure_main_window(app: &AppHandle) -> Result<WebviewWindow, String> {
    // Return existing window if available
    if let Some(window) = app.get_webview_window(MAIN_WINDOW_LABEL) {
        return Ok(window);
    }
    
    // Create new window
    WebviewWindowBuilder::new(
        app,
        MAIN_WINDOW_LABEL,
        WebviewUrl::App("index.html".into()),
    )
    .title("Ownstash Downloader")
    .inner_size(1280.0, 800.0)
    .min_inner_size(900.0, 600.0)
    .resizable(true)
    .center()
    .visible(false)
    .build()
    .map_err(|e| format!("Failed to create window: {}", e))
}

pub fn show_main_window(app: &AppHandle) {
    match ensure_main_window(app) {
        Ok(window) => {
            let _ = window.unminimize();
            let _ = window.show();
            let _ = window.set_focus();
        }
        Err(err) => {
            eprintln!("[Window] {}", err);
        }
    }
}

fn close_main_window(app: &AppHandle) {
    if let Some(window) = app.get_webview_window(MAIN_WINDOW_LABEL) {
        let _ = window.close();
    }
}
```

### Update Progress

```rust
pub fn set_taskbar_progress(
    app_handle: &AppHandle,
    progress: f64,  // 0.0 to 1.0
    state: ProgressState,
) {
    #[cfg(target_os = "windows")]
    {
        use windows::Win32::Foundation::HWND;
        
        let window = app_handle.get_webview_window("main").unwrap();
        let hwnd = HWND(window.hwnd().unwrap().0 as isize);
        
        unsafe {
            let taskbar = get_taskbar_interface();
            
            match state {
                ProgressState::Normal => {
                    taskbar.SetProgressState(hwnd, TBPF_NORMAL);
                    let completed = (progress * 100.0) as u64;
                    taskbar.SetProgressValue(hwnd, completed, 100);
                }
                ProgressState::Error => {
                    taskbar.SetProgressState(hwnd, TBPF_ERROR);
                }
                ProgressState::Paused => {
                    taskbar.SetProgressState(hwnd, TBPF_PAUSED);
                }
                ProgressState::Indeterminate => {
                    taskbar.SetProgressState(hwnd, TBPF_INDETERMINATE);
                }
                ProgressState::NoProgress => {
                    taskbar.SetProgressState(hwnd, TBPF_NOPROGRESS);
                }
            }
        }
    }
    
    #[cfg(target_os = "macos")]
    {
        // Update dock icon progress
        use cocoa::appkit::{NSApp, NSApplication};
        use cocoa::base::nil;
        
        unsafe {
            let app = NSApp();
            if progress > 0.0 && progress < 1.0 {
                let badge = format!("{:.0}%", progress * 100.0);
                let badge_str = NSString::alloc(nil)
                    .init_str(&badge);
                app.dockTile().setBadgeLabel_(badge_str);
            } else {
                app.dockTile().setBadgeLabel_(nil);
            }
        }
    }
}
```

### Frontend Integration

```tsx
// Update taskbar progress based on downloads
useEffect(() => {
  const activeDownloads = downloads.filter(
    d => d.status === 'downloading'
  );
  
  if (activeDownloads.length === 0) {
    // Clear progress
    invoke('set_taskbar_progress', {
      progress: 0,
      state: 'NoProgress'
    });
    return;
  }
  
  // Calculate overall progress
  const totalProgress = activeDownloads.reduce(
    (sum, d) => sum + (d.progress || 0), 0
  ) / activeDownloads.length;
  
  invoke('set_taskbar_progress', {
    progress: totalProgress / 100,
    state: 'Normal'
  });
}, [downloads]);
```

## Auto-Start on Login

Launch Ownstash automatically when you log in.

```rust
use tauri_plugin_autostart::{ManagerExt, MacosLauncher};

.plugin(
    tauri_plugin_autostart::init(
        MacosLauncher::LaunchAgent, 
        Some(vec!["--minimized"])
    )
)
.setup(move |app| {
    // Enable autostart by default
    let app_state = app.state::<AppState>();
    if let Ok(db) = app_state.db.lock() {
        if db.get_setting("autostart_initialized").unwrap_or(None).is_none() {
            let autostart_manager = app.autolaunch();
            let _ = autostart_manager.enable();
            let _ = db.save_setting("autostart_initialized", "true");
            let _ = db.save_setting("autostart_enabled", "true");
        }
    }
    
    Ok(())
})
```

## Deep Link Protocol Handler

Register custom URL protocol (`ownstash://`):

```rust
use tauri_plugin_deep_link::DeepLinkExt;

.plugin(tauri_plugin_deep_link::init())
.setup(|app| {
    // Register protocol handler
    #[cfg(any(target_os = "linux", all(debug_assertions, windows)))]
    {
        let _ = app.deep_link().register_all();
    }
    
    // Listen for deep link events
    let handle = app.handle().clone();
    app.listen("deep-link://new-url", move |event| {
        let payload = event.payload();
        
        if payload.contains("auth") {
            // OAuth callback
            let _ = handle.emit("oauth-deep-link", payload);
        } else if let Some(url) = parse_deep_link(payload) {
            // Download request
            let _ = handle.emit("extension-download-request", &url);
        }
    });
    
    Ok(())
})
```

**Parse Deep Link:**

```rust
fn parse_deep_link(url: &str) -> Option<String> {
    let parsed = url::Url::parse(url).ok()?;
    
    if parsed.scheme() != "ownstash" {
        return None;
    }
    
    match parsed.host_str()? {
        "download" | "vault-download" => {
            parsed.query_pairs()
                .find(|(key, _)| key == "url")
                .map(|(_, value)| value.to_string())
        }
        _ => None
    }
}
```

## Background Mode

Keep downloads running when window is closed:

```rust
let background_mode = Arc::new(AtomicBool::new(false));

// Check if started with --minimized flag
let args: Vec<String> = std::env::args().collect();
if args.contains(&"--minimized".to_string()) {
    background_mode.store(true, Ordering::SeqCst);
    close_main_window(&app_handle);
} else {
    background_mode.store(false, Ordering::SeqCst);
    show_main_window(&app_handle);
}

// Prevent exit while downloads active
.build(tauri::generate_context!())
.expect("error while building tauri application")
.run(move |app_handle, event| match event {
    RunEvent::ExitRequested { api, .. } => {
        let has_active_downloads = check_active_downloads();
        let is_background = background_mode.load(Ordering::SeqCst);
        
        if has_active_downloads && !is_background {
            api.prevent_exit();
            toast::show("Downloads in progress. Minimizing to tray...");
        }
    }
    _ => {}
});
```

## Troubleshooting

### Tray Icon Not Showing

1. **Restart the app** to reinitialize tray
2. **Check system tray settings** - icon may be hidden
3. **Reinstall app** if icon is corrupted

### Notifications Not Working

1. **Check OS permissions** - Grant notification access
2. **Verify app settings** - Ensure notifications are enabled
3. **Test notification** - Use test button in settings

### Autostart Not Working

1. **Check startup apps** in system settings
2. **Reinstall app** to re-register autostart
3. **Manually add** to startup if automatic fails
