## Overview

The Vault feature provides military-grade encryption for your sensitive downloads. Files are encrypted locally using AES-256-GCM with a PIN-derived key, then optionally synced to Google Drive in encrypted form.

<Warning>
**Zero-knowledge architecture:** Your PIN never leaves your device. Even if your Google Drive is compromised, your files remain secure without the PIN.
</Warning>

## Security Architecture

### Encryption Standard

**AES-256-GCM** (Advanced Encryption Standard with Galois/Counter Mode)

- **256-bit key** derived from your PIN using Argon2
- **Authenticated encryption** - detects tampering
- **Unique nonce per chunk** - prevents replay attacks
- **Streaming encryption** - handles large files efficiently

```rust
use aes_gcm::{
    aead::{Aead, KeyInit, OsRng},
    Aes256Gcm, Nonce,
};
use argon2::{Argon2, PasswordHasher};

const NONCE_SIZE: usize = 12;
const KEY_SIZE: usize = 32;  // 256 bits
const ENCRYPTED_EXTENSION: &str = ".slasshy";
```

### PIN Security

<Steps>
  <Step title="PIN Hashing (Argon2)">
    Your PIN is hashed using Argon2 - the winner of the Password Hashing Competition:
    
    ```rust
    pub fn vault_setup(pin: String) -> Result<()> {
        // Generate random salt
        let salt = SaltString::generate(&mut OsRng);
        
        // Hash PIN with Argon2
        let argon2 = Argon2::default();
        let pin_hash = argon2
            .hash_password(pin.as_bytes(), &salt)?
            .to_string();
        
        // Store hash and salt (NOT the PIN)
        let config = VaultConfig {
            pin_hash,
            salt: salt.to_string(),
            created_at: Utc::now().timestamp(),
        };
        
        save_vault_config(&config)?;
    }
    ```
  </Step>
  
  <Step title="Key Derivation">
    When you unlock the vault, your PIN derives the encryption key:
    
    ```rust
    fn derive_key_from_pin(pin: &str, salt: &[u8]) -> [u8; KEY_SIZE] {
        let mut key = [0u8; KEY_SIZE];
        
        Argon2::default()
            .hash_password_into(pin.as_bytes(), salt, &mut key)
            .expect("Failed to derive key");
        
        key  // 256-bit encryption key
    }
    ```
  </Step>
  
  <Step title="Session Management">
    The key is held in memory while the vault is unlocked:
    
    ```rust
    lazy_static! {
        static ref VAULT_SESSION: Mutex<Option<VaultSession>> = 
            Mutex::new(None);
    }
    
    struct VaultSession {
        key: [u8; KEY_SIZE],
        unlocked_at: i64,
    }
    ```
  </Step>
</Steps>

pub fn vault_setup(
    app_handle: AppHandle, 
    pin: String
) -> Result<(), String> {
    if pin.len() < 4 {
        return Err("PIN must be at least 4 digits".to_string());
    }
    
    // Generate salt and hash PIN
    let salt = SaltString::generate(&mut OsRng);
    let argon2 = Argon2::default();
    let pin_hash = argon2
        .hash_password(pin.as_bytes(), &salt)?
        .to_string();
    
    let config = VaultConfig {
        pin_hash,
        salt: salt.to_string(),
        created_at: chrono::Utc::now().timestamp(),
        last_accessed: None,
    };
    
    // Create vault directories
    let files_dir = get_vault_files_dir(&app_handle);
    fs::create_dir_all(&files_dir)?;
    
    save_vault_config(&app_handle, &config)?;
    
    // Auto-unlock after setup
    let key = derive_key_from_pin(&pin, salt.as_str().as_bytes());
    *VAULT_SESSION.lock().unwrap() = Some(VaultSession { key });
    
    Ok(())
}

## File Encryption

### Chunked Streaming Encryption

Large files are encrypted in 1MB chunks to handle any file size:

```rust
fn encrypt_file(
    key: &[u8; KEY_SIZE], 
    input_path: &Path, 
    output_path: &Path
) -> Result<(), String> {
    const CHUNK_SIZE: usize = 1024 * 1024;  // 1MB
    const VAULT_MAGIC: &[u8; 4] = b"SLV2";
    
    let cipher = Aes256Gcm::new_from_slice(key)?;
    
    // Generate random base nonce
    let mut base_nonce = [0u8; NONCE_SIZE];
    OsRng.fill_bytes(&mut base_nonce);
    
    let mut input_file = File::open(input_path)?;
    let file_size = input_file.metadata()?.len();
    let mut output_file = File::create(output_path)?;
    
    // Write header: [MAGIC][nonce][file_size]
    output_file.write_all(VAULT_MAGIC)?;
    output_file.write_all(&base_nonce)?;
    output_file.write_all(&file_size.to_le_bytes())?;
    
    let mut buffer = vec![0u8; CHUNK_SIZE];
    let mut chunk_index: u64 = 0;
    
    loop {
        let bytes_read = input_file.read(&mut buffer)?;
        if bytes_read == 0 { break; }
        
        // Derive unique nonce for this chunk
        let mut chunk_nonce = base_nonce;
        let index_bytes = chunk_index.to_le_bytes();
        for i in 0..8 {
            chunk_nonce[i] ^= index_bytes[i];
        }
        let nonce = Nonce::from_slice(&chunk_nonce);
        
        // Encrypt chunk
        let plaintext = &buffer[..bytes_read];
        let ciphertext = cipher.encrypt(nonce, plaintext)?;
        
        // Write [chunk_size][ciphertext]
        let chunk_len = ciphertext.len() as u32;
        output_file.write_all(&chunk_len.to_le_bytes())?;
        output_file.write_all(&ciphertext)?;
        
        chunk_index += 1;
    }
    
    Ok(())
}
```

### File Format

```
.slasshy File Structure:

[Header - 24 bytes]
- Magic: "SLV2" (4 bytes)
- Base Nonce: 12 bytes
- File Size: 8 bytes (little-endian)

[Chunks]
Repeat for each 1MB chunk:
- Chunk Length: 4 bytes (little-endian)
- Encrypted Data: Variable (includes 16-byte auth tag)
```

```rust
#[tauri::command]
pub async fn vault_add_file(
    app_handle: AppHandle,
    source_path: String,
    original_name: String,
    file_type: String,
    thumbnail: Option<String>,
    delete_original: bool,
) -> Result<VaultFile, String> {
    // Get encryption key
    let key = get_vault_key()?;
    
    let source = PathBuf::from(&source_path);
    let file_size = fs::metadata(&source)?.len();
    
    // Generate encrypted filename
    let file_id = uuid::Uuid::new_v4().to_string();
    let encrypted_name = format!("{}.slasshy", file_id);
    let dest_path = get_vault_files_dir(&app_handle).join(&encrypted_name);
    
    // Encrypt in background thread
    tokio::task::spawn_blocking(move || {
        encrypt_file(&key, &source, &dest_path)
    }).await??;
    
    // Delete original if requested
    if delete_original {
        fs::remove_file(&source)?;
    }
    
    Ok(VaultFile {
        id: file_id,
        original_name,
        encrypted_name,
        size_bytes: file_size,
        added_at: Utc::now().timestamp(),
        file_type,
        thumbnail,
        is_folder: false,
        folder_entries: None,
    })
}
```

## Decryption & Export

### Decrypt for Playback

Create temporary decrypted file for in-app playback:

```rust
#[tauri::command]
pub async fn vault_get_temp_playback_path(
    app_handle: AppHandle,
    file_id: String,
    encrypted_name: String,
    original_name: String,
) -> Result<String, String> {
    let key = get_vault_key()?;
    
    let encrypted_path = get_vault_files_dir(&app_handle)
        .join(&encrypted_name);
    
    // Create temp directory inside vault (more secure)
    let temp_dir = get_vault_dir(&app_handle).join("temp");
    fs::create_dir_all(&temp_dir)?;
    
    // Generate random temp filename (no traces of original name)
    let file_extension = PathBuf::from(&original_name)
        .extension()
        .map(|e| format!(".{}", e.to_string_lossy()))
        .unwrap_or_else(|| ".tmp".to_string());
    
    let temp_file_id = uuid::Uuid::new_v4().to_string();
    let temp_path = temp_dir.join(format!("{}{}", temp_file_id, file_extension));
    
    // Decrypt in background
    tokio::task::spawn_blocking(move || {
        decrypt_file(&key, &encrypted_path, &temp_path)
    }).await??;
    
    Ok(temp_path.to_string_lossy().to_string())
}
```

## PIN Management

### Change PIN

Re-encrypt all vault files with a new PIN:

```rust
#[tauri::command]
pub fn vault_change_pin(
    app_handle: AppHandle,
    current_pin: String,
    new_pin: String,
) -> Result<(), String> {
    // Verify current PIN
    let config = load_vault_config(&app_handle)?;
    let parsed_hash = PasswordHash::new(&config.pin_hash)?;
    Argon2::default()
        .verify_password(current_pin.as_bytes(), &parsed_hash)
        .map_err(|_| "Current PIN is incorrect")?;
    
    // Derive both keys
    let current_key = derive_key_from_pin(&current_pin, config.salt.as_bytes());
    let new_salt = SaltString::generate(&mut OsRng);
    let new_key = derive_key_from_pin(&new_pin, new_salt.as_str().as_bytes());
    
    // Get all encrypted files
    let files_dir = get_vault_files_dir(&app_handle);
    let vault_files: Vec<_> = fs::read_dir(&files_dir)?
        .filter_map(|e| e.ok())
        .filter(|e| e.path().extension() == Some(OsStr::new("slasshy")))
        .collect();
    
    let temp_dir = get_vault_dir(&app_handle).join("reencrypt_temp");
    fs::create_dir_all(&temp_dir)?;
    
    // Re-encrypt each file
    for entry in vault_files {
        let encrypted_path = entry.path();
        let temp_decrypted = temp_dir.join(format!("{}_dec", entry.file_name()));
        let temp_reencrypted = temp_dir.join(format!("{}_new", entry.file_name()));
        
        // Decrypt with old key
        decrypt_file(&current_key, &encrypted_path, &temp_decrypted)?;
        
        // Re-encrypt with new key
        encrypt_file(&new_key, &temp_decrypted, &temp_reencrypted)?;
        
        // Replace original
        fs::rename(&temp_reencrypted, &encrypted_path)?;
        fs::remove_file(&temp_decrypted)?;
    }
    
    fs::remove_dir_all(&temp_dir)?;
    
    // Update config with new PIN hash
    let new_pin_hash = Argon2::default()
        .hash_password(new_pin.as_bytes(), &new_salt)?
        .to_string();
    
    let new_config = VaultConfig {
        pin_hash: new_pin_hash,
        salt: new_salt.to_string(),
        created_at: config.created_at,
        last_accessed: Some(Utc::now().timestamp()),
    };
    save_vault_config(&app_handle, &new_config)?;
    
    // Update session with new key
    *VAULT_SESSION.lock().unwrap() = Some(VaultSession {
        key: new_key,
        unlocked_at: Utc::now().timestamp(),
    });
    
    Ok(())
}
```

### Reset Vault

<Warning>
**DESTRUCTIVE ACTION:** This permanently deletes all encrypted files. Cannot be undone.
</Warning>

```rust
#[tauri::command]
pub fn vault_reset(
    app_handle: AppHandle, 
    pin: String
) -> Result<(), String> {
    // Verify PIN before reset
    let config = load_vault_config(&app_handle)?;
    let parsed_hash = PasswordHash::new(&config.pin_hash)?;
    Argon2::default()
        .verify_password(pin.as_bytes(), &parsed_hash)
        .map_err(|_| "Invalid PIN")?;
    
    // Lock vault
    *VAULT_SESSION.lock().unwrap() = None;
    
    // Delete entire vault directory
    let vault_dir = get_vault_dir(&app_handle);
    if vault_dir.exists() {
        fs::remove_dir_all(&vault_dir)?;
    }
    
    Ok(())
}
```

## Folder Support

Encrypt entire folders as ZIP archives:

```rust
#[tauri::command]
pub async fn vault_add_folder(
    app_handle: AppHandle,
    folder_path: String,
    folder_name: String,
    delete_original: bool,
) -> Result<VaultFile, String> {
    let key = get_vault_key()?;
    let source_dir = PathBuf::from(&folder_path);
    
    // Collect folder entries
    let mut folder_entries = Vec::new();
    let mut total_size = 0u64;
    
    for entry in WalkDir::new(&source_dir).into_iter().filter_map(|e| e.ok()) {
        let path = entry.path();
        let relative_path = path.strip_prefix(&source_dir)?
            .to_string_lossy()
            .to_string();
        
        let size = if path.is_file() { 
            path.metadata()?.len()
        } else { 0 };
        
        total_size += size;
        
        folder_entries.push(VaultFolderEntry {
            name: path.file_name()?.to_string_lossy().to_string(),
            path: relative_path,
            size_bytes: size,
            file_type: detect_file_type(path),
            is_directory: path.is_dir(),
        });
    }
    
    // Create ZIP archive
    let temp_dir = get_vault_dir(&app_handle).join("temp");
    let file_id = uuid::Uuid::new_v4().to_string();
    let temp_zip = temp_dir.join(format!("{}.zip", file_id));
    
    tokio::task::spawn_blocking(move || {
        let zip_file = File::create(&temp_zip)?;
        let mut zip = ZipWriter::new(zip_file);
        
        for entry in &folder_entries {
            if entry.is_directory {
                zip.add_directory(&entry.path, Default::default())?;
            } else {
                let full_path = source_dir.join(&entry.path);
                zip.start_file(&entry.path, Default::default())?;
                let mut file = File::open(&full_path)?;
                std::io::copy(&mut file, &mut zip)?;
            }
        }
        
        zip.finish()?;
        Ok::<_, String>(())
    }).await??;
    
    // Encrypt the ZIP
    let encrypted_name = format!("{}.slasshy", file_id);
    let dest_path = get_vault_files_dir(&app_handle).join(&encrypted_name);
    
    tokio::task::spawn_blocking(move || {
        encrypt_file(&key, &temp_zip, &dest_path)
    }).await??;
    
    fs::remove_file(&temp_zip)?;
    
    if delete_original {
        fs::remove_dir_all(&source_dir)?;
    }
    
    Ok(VaultFile {
        id: file_id,
        original_name: folder_name,
        encrypted_name,
        size_bytes: total_size,
        added_at: Utc::now().timestamp(),
        file_type: "folder".to_string(),
        thumbnail: None,
        is_folder: true,
        folder_entries: Some(folder_entries),
    })
}
```

## Troubleshooting

### Forgot PIN

<Warning>
**There is NO way to recover files without the correct PIN.** This is by design - zero-knowledge encryption.
</Warning>

Options:
1. Try common PINs you've used
2. If you have cloud backup, the PIN is the same on all devices
3. As a last resort, use `vault_reset()` to start fresh (loses all files)

### Decryption Failed

```
Error: "Decryption failed - invalid PIN or corrupted file"
```

**Causes:**
- Wrong PIN entered
- File was corrupted during transfer
- File was modified/tampered with

**Solutions:**
1. Verify PIN is correct
2. Re-download file from cloud backup
3. Check file integrity with cloud hash

### Cloud Sync Conflicts

If two devices modified vault simultaneously:

```typescript
// Merge strategy: Keep newer files
const mergeVaultIndexes = (
  local: CloudVaultIndex, 
  remote: CloudVaultIndex
) => {
  const merged = new Map<string, VaultFile>();
  
  // Add all local files
  local.files.forEach(f => merged.set(f.id, f));
  
  // Add remote files (overwrite if newer)
  remote.files.forEach(f => {
    const existing = merged.get(f.id);
    if (!existing || f.added_at > existing.added_at) {
      merged.set(f.id, f);
    }
  });
  
  return Array.from(merged.values());
};
```
