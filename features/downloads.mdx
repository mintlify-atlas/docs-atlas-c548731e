## Overview

Ownstash Downloader supports downloading media from over 1000 platforms using the powerful yt-dlp engine. Get videos, audio, and playlists with advanced quality controls and format options.

<Note>
Ownstash automatically detects the best download engine for each URL - SNDE for direct files, Media Engine for adaptive streaming, or yt-dlp for platforms.
</Note>

## Supported Platforms

Ownstash works with any site supported by yt-dlp, including:

- **Video:** YouTube, Vimeo, Dailymotion, TikTok, Instagram, Twitter/X
- **Audio:** SoundCloud, Spotify (via API), Bandcamp, Mixcloud
- **Live Streaming:** Twitch, YouTube Live, Facebook Live
- **Education:** Udemy, Coursera, Khan Academy
- **And 1000+ more platforms**

## Download Engines

Ownstash automatically routes downloads to the optimal engine:

<Accordion title="SNDE ACCELERATED">
**Best for:** Direct file downloads with range request support

- **Multi-connection downloads** (up to 8 parallel streams)
- **Resume capability** for interrupted downloads  
- **Optimal for:** Cloud storage (Google Drive, Dropbox, OneDrive)

```rust
// SNDE routes files based on probe results
if routing_decision.probe_result.supports_range {
    let snde_request = SNDERequest {
        url: url.clone(),
        recommended_connections: 8,
    };
    SNDE_ENGINE.download(snde_request).await;
}
```
</Accordion>

<Accordion title="SNDE SAFE">
**Best for:** Servers without range request support

- **Single connection** download
- **Reliable** for all file types
- **Fallback mode** when acceleration isn't supported
</Accordion>

<Accordion title="MEDIA ENGINE">
**Best for:** Streaming platforms (YouTube, Vimeo, etc.)

- **Adaptive streaming** support (DASH, HLS)
- **Quality selection** with format merging
- **Subtitle embedding** and metadata extraction

```rust
// yt-dlp with optimized settings
let mut args = vec![
    "--concurrent-fragments", "4",
    "--retries", "4",
    "-f", "bestvideo+bestaudio/best",
];

if audio_only {
    args.extend(["-x", "--audio-format", "mp3"]);
}
```
</Accordion>

## Progress Tracking

Real-time updates appear on the Downloads page:

```tsx
// Subscribe to progress events
useEffect(() => {
  const unlisten = api.onDownloadProgress((progress) => {
    setDownloads(prev => prev.map(d => 
      d.id === progress.id 
        ? { ...d, 
            progress: progress.progress,
            speed: progress.speed,
            eta: progress.eta,
            engine_badge: progress.engine_badge
          }
        : d
    ));
  });
  
  return () => unlisten();
}, []);
```

**Progress UI Components:**

- **Progress bar** with percentage (0-100%)
- **Download speed** (e.g., "2.5 MiB/s")
- **ETA** (estimated time remaining)
- **Engine badge** (SNDE ACCELERATED, MEDIA ENGINE)
- **Status indicator** (downloading, completed, failed)

<Warning>
Don't close the app while downloads are active. Ownstash can minimize to the system tray to continue downloads in the background.
</Warning>

### Direct Files

Cloud storage and direct file links:

```rust
// Probe file metadata
pub async fn probe_direct_file(url: String) -> Result<DirectFileInfo> {
    let response = client.head(&url).send().await?;
    
    Ok(DirectFileInfo {
        file_size: response.headers()
            .get(CONTENT_LENGTH)
            .and_then(|v| v.to_str().ok())
            .and_then(|s| s.parse().ok())
            .unwrap_or(0),
        filename: extract_filename_from_headers(&response),
        supports_range: response.headers()
            .get("accept-ranges")
            .map(|v| v == "bytes")
            .unwrap_or(false),
    })
}
```

## Troubleshooting

### Download Failed

1. **Check yt-dlp version** - Go to Settings > Update yt-dlp
2. **Try different quality** - Some formats may be unavailable
3. **Check internet connection** - Ensure stable connection
4. **Platform restrictions** - Some content may be geo-blocked

### Slow Downloads

- **Use SNDE ACCELERATED** for direct files (automatic)
- **Check concurrent downloads** - Limit active downloads to 2-3
- **Network throttling** - Some platforms limit download speed

### Format Not Available

```tsx
// Fallback to best available format
const formatSelector = quality === '4k' 
  ? 'bestvideo[height<=2160]+bestaudio/best'
  : 'bestvideo+bestaudio/best';
```
